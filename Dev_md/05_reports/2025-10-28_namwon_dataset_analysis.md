## 🎯 개별 데이터셋 상세 분석

### 1. 🏠 dataset_greenhouse_multi (다중 클래스 온실 탐지)

#### 📊 데이터 구성
```
dataset_greenhouse_multi/
├── 📄 dataset.yaml          # YOLO 데이터셋 설정
├── 📄 dataset_stats.json    # 데이터셋 통계
├── 📁 images/
│   ├── 📁 train/            # 훈련용 이미지 (497개)
│   ├── 📁 val/              # 검증용 이미지 (52개)
│   └── 📁 test/             # 테스트용 이미지 (55개)
└── 📁 labels/
    ├── 📁 train/            # 훈련용 라벨 (497개)
    ├── 📁 val/              # 검증용 라벨 (52개)
    ├── 📁 test/             # 테스트용 라벨 (55개)
    ├── 📄 train.cache       # YOLO 캐시 파일
    └── 📄 val.cache         # YOLO 캐시 파일
```

#### 🎯 클래스 정보
- **클래스 수**: 다중 온실 타입 (설정 파일에서 확인 필요)
- **이미지 해상도**: 고해상도 TIF 형식
- **라벨 형식**: YOLO bbox 형식 (class_id x_center y_center width height)

#### 📈 데이터 분할 비율
- **Train**: 497개 (82.3%)
- **Validation**: 52개 (8.6%)
- **Test**: 55개 (9.1%)
- **총합**: 604개 이미지
